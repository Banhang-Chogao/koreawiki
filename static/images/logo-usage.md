# KoreaWiki Logo Usage Guidelines

## Logo Files

| File | Usage | Format |
|------|-------|--------|
| `logo.svg` | Primary logo — light backgrounds | SVG |
| `logo-dark.svg` | Dark mode / dark backgrounds | SVG |
| `favicon.svg` | Browser tab icon, 16px+ | SVG |
| `apple-touch-icon.svg` | iOS home screen, 180×180 | SVG (convert to PNG for production) |
| `favicon.ico` | Legacy browsers | ICO (convert from SVG) |

## Logo Structure

- **Icon**: Teal (#00B5A6) rounded square containing a white document with folded corner — symbolizing knowledge, wiki content, and Korean minimalist aesthetics.
- **Wordmark**: "KoreaWiki" set in Pretendard or system sans-serif (bold, 800 weight, slightly condensed tracking).
- **Color split**: "Korea" in primary text color, "Wiki" in teal (#00B5A6).

## Clear Space

Maintain clear space equal to **half the icon width** on all sides of the logo. No text, graphics, or other elements may intrude into this zone.

## Minimum Size

- Full logo: **120px wide** (web) / **1.2 inches** (print)
- Icon only: **24px** (web) / **0.25 inches** (print)

## Color Usage

### Light Background (logo.svg)
- Icon container: #00B5A6
- "Korea": #1a1a2e (dark navy)
- "Wiki": #00B5A6

### Dark Background (logo-dark.svg)
- Icon container: #00B5A6
- "Korea": #ffffff
- "Wiki": #00B5A6

### Monochrome (special cases only)
Use solid #1a1a2e on light or #ffffff on dark. Never use the teal color in monochrome contexts.

## Don'ts

- ❌ Do not stretch, skew, rotate, or distort the logo
- ❌ Do not change the icon color to anything other than teal (#00B5A6)
- ❌ Do not rearrange the icon and wordmark positions
- ❌ Do not use the logo on low-contrast backgrounds
- ❌ Do not add shadows, gradients, or effects unless specified
- ❌ Do not recreate the logo using different fonts
- ❌ Do not separate the icon from the wordmark (except for favicon/avatar use)

## Background Compatibility

| Background | Logo to use |
|------------|-------------|
| White / light (#f8f9fa) | `logo.svg` |
| Dark (#1a1a2e, #000) | `logo-dark.svg` |
| Photos / busy patterns | Do not use — use solid background panel instead |
| Teal (#00B5A6) | Use inverted version (white icon + white text) |

## Favicon / Avatar

For favicon, app icon, or avatar use cases, the icon mark may be used alone (without the wordmark). Always use the full-color teal version.

## File Format

Always prefer **SVG** for web use. Raster images (PNG, ICO) should only be used where SVG is not supported. Generate raster assets from the SVG source at the required resolution.
